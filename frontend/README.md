# Telegram Mini App - Онлайн-консультации

## Структура проекта

```
frontend/
├── index.html          # Главная страница (Dashboard)
├── doctor.html         # Страница врача (заглушка)
├── profile.html        # Профиль (заглушка)
├── css/
│   └── style.css      # Все стили приложения
├── js/
│   ├── app.js         # Основная логика
│   └── data.js        # Данные врачей
└── img/
    └── doctors/       # Фото врачей
        └── arbatov.jpg  # Фото Вячеслава Арбатова
```

## Как запустить локально

1. **Добавьте фото врача:**
   - Сохраните фото Вячеслава Арбатова в папку `img/doctors/` с именем `arbatov.jpg`

2. **Откройте в браузере:**
   - Просто откройте файл `index.html` в браузере (Chrome/Safari)
   - Или запустите локальный сервер (см. ниже)

3. **Запуск через локальный сервер (рекомендуется):**
   ```bash
   cd /Users/miroslavtito/tgapp1/frontend
   python3 -m http.server 8000
   ```
   Затем откройте в браузере: http://localhost:8000
